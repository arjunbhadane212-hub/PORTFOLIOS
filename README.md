# Portfolio Website

A clean, modern, multi-page portfolio website designed for creators, freelancers, and small projects.  
Built with simple, maintainable HTML and CSS and hosted using GitHub Pages.

## Features
- Multi-page layout (Home, About, Work, Pricing, Contact)
- Responsive design (desktop & mobile)
- GitHub-inspired dark theme
- Easy customization for colors, content, and sections
- Free hosting via GitHub Pages

## My Role
- Design direction and layout structure
- UI/UX decisions and page organization
- Custom styling and responsiveness
- Project setup and deployment

Code implementation was assisted using AI tools, with final structure, design, and customization handled by me.

## Customization
This template can be duplicated and customized per client:
- Change colors and branding
- Add or remove pages
- Insert images or project screenshots
- Adjust pricing and content
